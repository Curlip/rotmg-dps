Curlip's DPS Calculator

o/

Welcome to a DPS calculating project I have been working on recently. I'm sure anyone who is here has used / heard about Pfiffle DPS Calculator, which arguably has many more features / modes. One issue with Pfiffle's approach to his DPS Calculator is that it requires manual updates. The last of these being near the start of the year.

I have tried very hard to make my calculator auto-updating. Using the data off static.drips.pw means I can call my calculator up to date almost immediently after the game updates. It also mean testing leaks too...
